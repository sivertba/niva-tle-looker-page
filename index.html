Satellite Forecaster

The forecast is generated using the pyorbital library. The forecast is generated for the next week and is updated every day. The cloud cover is retrieved from the Norwegian Meteorological Institute. The cloud cover is given as the median of a grid at the location.The forecast is generated using the following parameters:
